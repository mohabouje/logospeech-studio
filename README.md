
# LogoSpeech Studio. #

A new multi platform tool (Windows, Ubuntu \& Mac) that provides *visual support for the language learning of people with hearing impairments.* The tool use some of the  advantage speech signal processing technologies (Spectral Power Density, Spectrogram, melody, AZCR, Pitch, LPC Filtering ...)  to provide a suitable visual way of the main acoustic properties.

### ¿ Qué diferencia a LogoSpeech? ###

LogoSpeech Studio es un proyecto de software libre multiplataforma desarrollado en C/C++ con las librerias gráficas de Qt. Nace con la finalidad de servir como heraamienta de entrenamiento para el reconocimiento del habla y correcta vocalización para personas con problemas auditivos.

Se basa en el análisis temporal/espectral de la señal del habla para la extración de parámetros, entre los más destacados:




* Energy estimation
* Average Zero Cross Rate.
* Pitch estimation
* Melody estimation
* Spectrogram.
* LPC Spectrogram.
* Cepstrum.
* Bank FIlters.
